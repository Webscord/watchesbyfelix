Documentation du site WatchesByFelix

Cette documentation explique comment mettre en ligne et maintenir le site web WatchesByFelix sur GitHub Pages.

1. Structure du site

Le site est organisé selon la structure suivante :

watchesbyfelix/
├── index.html              # Page d'accueil
├── shop.html               # Page boutique avec badges VENDU
├── gallery.html            # Galerie de photos
├── ebook.html              # Page de l'ebook
├── about.html              # Page À propos
├── contact.html            # Page Contact
├── privacy.html            # Politique de confidentialité
├── terms.html              # Conditions générales de vente
├── shipping.html           # Livraison et retours
├── legal.html              # Mentions légales
├── assets/
│   ├── css/
│   │   ├── style.css       # Styles principaux
│   │   ├── shop.css        # Styles spécifiques à la boutique
│   │   └── additional.css  # Styles additionnels
│   ├── js/
│   │   ├── main.js         # Script principal
│   │   └── shop.js         # Script pour la boutique
│   └── images/
│       ├── logo.svg        # Logo principal
│       ├── logo-white.svg  # Logo blanc pour le footer
│       ├── favicon.ico     # Favicon
│       ├── og-image.jpg    # Image pour partage social
│       └── ...             # Autres images
└── README.md               # Documentation

2. Mise en ligne sur GitHub Pages

2.1 Création du dépôt GitHub

  1. Connectez-vous à votre compte GitHub
  2. Cliquez sur "New repository" pour créer un nouveau dépôt
  3. Nommez le dépôt watchesbyfelix.github.io (ce format spécifique est nécessaire pour GitHub Pages)
  4. Choisissez "Public" comme visibilité
  5. Cliquez sur "Create repository"

2.2 Téléchargement des fichiers

Vous pouvez télécharger les fichiers sur GitHub de deux façons :

Option 1 : Via l'interface web de GitHub

  1. Accédez à votre dépôt watchesbyfelix.github.io
  2. Cliquez sur "Add file" puis "Upload files"
  3. Faites glisser tous les fichiers et dossiers du site
  4. Ajoutez un message de commit comme "Initial upload"
  5. Cliquez sur "Commit changes"

Option 2 : Via Git en ligne de commande

Si vous êtes à l'aise avec Git, vous pouvez utiliser les commandes suivantes :

git clone https://github.com/votre-nom-utilisateur/watchesbyfelix.github.io.git
cd watchesbyfelix.github.io
# Copiez tous les fichiers du site dans ce dossier
git add .
git commit -m "Initial commit"
git push origin main

2.3 Activation de GitHub Pages

  1. Accédez à votre dépôt sur GitHub
  2. Cliquez sur "Settings" (onglet en haut à droite)
  3. Dans le menu de gauche, cliquez sur "Pages"
  4. Dans la section "Source", sélectionnez "main" comme branche
  5. Cliquez sur "Save"
  6. Attendez quelques minutes que GitHub déploie votre site
  7. Votre site sera accessible à l'adresse https://watchesbyfelix.github.io

Note : Le déploiement initial peut prendre jusqu'à 10 minutes. Les mises à jour ultérieures sont généralement plus rapides.

3. Personnalisation du site

3.1 Modification des informations de contact

Les informations de contact sont présentes dans plusieurs fichiers :

  • Dans le pied de page de chaque page HTML (section footer)
  • Dans les pages légales (privacy.html, terms.html, shipping.html, legal.html)
  • Dans la page contact.html

Vous devez mettre à jour ces informations avec vos coordonnées réelles :

  • Adresse physique
  • Numéro de téléphone
  • Adresse email
  • Numéro d'entreprise du Québec (NEQ)

3.2 Ajout de vos propres images

Pour ajouter vos propres images de montres :

  1. Préparez des images de bonne qualité (idéalement au format 16:9 ou 4:3)
  2. Redimensionnez-les à une taille raisonnable (max 1200px de large)
  3. Placez-les dans le dossier assets/images/
  4. Mettez à jour les références dans les fichiers HTML

3.3 Modification des liens de la boutique

Dans le fichier shop.html, vous devez mettre à jour les liens de redirection vers votre site de vente externe :

<a href="https://external-shop-url.com/product1" class="shop-item-link" target="_blank">

Remplacez https://external-shop-url.com/product1 par l'URL réelle de votre produit sur le site de vente.

3.4 Mise à jour du statut des produits

Pour marquer un produit comme vendu :

  1. Localisez l'élément shop-item correspondant dans shop.html
  2. Changez l'attribut data-availability de "available" à "sold"
  3. Supprimez la balise <a> entourant le contenu
  4. Ajoutez la div sold-badge à l'intérieur de shop-item-image
  5. Mettez à jour le texte du statut

4. Maintenance du site

4.1 Mise à jour du contenu

Pour mettre à jour le contenu du site :

  1. Modifiez les fichiers HTML concernés
  2. Téléchargez les fichiers modifiés sur GitHub
  3. GitHub Pages déploiera automatiquement les modifications

4.2 Ajout de nouvelles pages

Pour ajouter une nouvelle page :

  1. Créez un nouveau fichier HTML en vous basant sur un fichier existant
  2. Assurez-vous d'inclure l'en-tête et le pied de page standard
  3. Ajoutez un lien vers cette page dans le menu de navigation
  4. Téléchargez le nouveau fichier sur GitHub

4.3 Mise à jour des métadonnées de partage social

Les métadonnées pour le partage sur les réseaux sociaux se trouvent dans la section <head> de chaque page HTML :

<meta property="og:title" content="WatchesByFelix | Montres Seiko Modifiées Artisanales">
<meta property="og:description" content="Découvrez des montres Seiko modifiées uniques, créées avec passion au Québec. Pièces exclusives, formations et communauté de passionnés.">
<meta property="og:image" content="https://watchesbyfelix.github.io/assets/images/og-image.jpg">
<meta property="og:url" content="https://watchesbyfelix.github.io">

Assurez-vous de mettre à jour ces informations pour chaque page, en particulier l'URL et le titre.

5. Conseils et bonnes pratiques

Conseil : Testez régulièrement votre site sur différents appareils (ordinateur, tablette, mobile) pour vous assurer qu'il s'affiche correctement.

Attention : N'oubliez pas de mettre à jour les informations légales (conditions générales, politique de confidentialité) en fonction de l'évolution de votre activité et des lois en vigueur.

5.1 Optimisation des images

Pour optimiser les performances de votre site :

  • Compressez vos images avant de les télécharger (utilisez des outils comme TinyPNG)
  • Utilisez des formats modernes comme WebP quand c'est possible
  • Spécifiez les dimensions des images dans le HTML pour éviter les décalages de mise en page

5.2 Référencement (SEO)

Pour améliorer le référencement de votre site :

  • Utilisez des titres descriptifs pour chaque page
  • Rédigez des méta-descriptions uniques et pertinentes
  • Utilisez des balises heading (h1, h2, h3) de manière hiérarchique
  • Ajoutez des attributs alt descriptifs à toutes vos images
  • Créez un fichier sitemap.xml et robots.txt
